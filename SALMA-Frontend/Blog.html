Abstract

This project presents an intelligent system for the automated grading of written exams and the generation of new exam questions, designed to streamline assessment processes and enhance educational efficiency. The system comprises two core components: automated grading and question generation. For the grading component, handwritten student responses are digitized using an optical character recognition (OCR) model, which accurately extracts textual data from scanned answer sheets. This text is then processed through a multi-agent system, which compares student responses to ideal answers using predefined grading criteria. The multi-agent architecture ensures robust and fair evaluation by leveraging diverse decision-making agents to evaluate different aspects of the answers, such as completeness, accuracy, and relevance.

The question generation component utilizes an agent-based framework to analyze provided educational material, such as lecture slides or textbooks in PDF format. The system identifies key concepts and themes within the content and generates a variety of question formats, including multiple-choice and short-answer questions. By integrating contextual understanding and multi-agent collaboration, the system creates well-structured exam questions aligned with the given material.

A significant aspect of this project is its focus on Arabic language support, alongside English. To enhance the performance of the Arabic OCR model, we aim to create the largest open-source handwritten Arabic dataset, with a size ten times larger than existing datasets. This dataset will contribute to advancing Arabic OCR capabilities and facilitate broader research in the field.

The proposed system is implemented using modern frameworks, including PyTorch and Node.js, ensuring scalability and ease of integration into existing academic workflows. This project demonstrates a practical application of AI in education, significantly reducing the time and effort required for exam grading and preparation while maintaining high standards of fairness and quality in assessments.

Introduction

This project aims to develop an intelligent system for the automated grading of written exams and the generation of exam questions, with a special focus on supporting handwritten Arabic text. The grading component will use advanced optical character recognition (OCR) models to digitize handwritten responses, which will then be analyzed by a multi-agent system to provide fair and accurate evaluations based on predefined criteria. The question generation component will employ an agent-based framework to create contextually relevant questions by analyzing educational content, such as PDFs or lecture materials. This system will support both Arabic and English, ensuring accessibility for diverse educational institutions.

The importance of this project lies in addressing critical challenges in educational assessment. Automated grading systems can significantly reduce the time and effort required by educators, enabling them to focus on teaching. Moreover, the system’s ability to generate exam questions aligned with course material ensures a consistent and efficient exam preparation process. With its support for handwritten Arabic text—a language currently underserved by existing AI systems—this project holds particular significance for the Arab world, where a substantial amount of academic content is written manually.

One groundbreaking aspect of this project is the development of the largest open-source handwritten Arabic dataset, with a size ten times larger than current datasets. This resource will not only enhance the accuracy of our OCR models but also contribute to the broader AI and research communities by addressing the scarcity of high-quality Arabic datasets.

By automating tedious processes, improving accessibility, and advancing AI capabilities in Arabic NLP and OCR, this project will fill a significant gap in educational technology and set a new standard for intelligent exam systems.

Methods

Our project employs a multi-faceted technical approach to build an intelligent system for automated exam grading and question generation. Below, we outline the methodologies and tools used:

  1. Optical Character Recognition (OCR):
    A deep learning-based OCR model will be trained to extract text from handwritten exam sheets. For Arabic text recognition, we aim to train a custom model using a Vision text transformer (TROCR). The model will be fine-tuned on the largest open-source Arabic handwritten dataset we plan to develop.
  2. Multi-Agent Grading System:
    Grading will be conducted using a multi-agent architecture, where individual agents analyze answers based on criteria such as correctness, structure, and completeness. The agents will use their ability to reason and understanding to compare extracted answers to ideal ones.
  3. Question Generation System:
    A multi-agent system will generate questions by analyzing input materials (PDFs, slides) using transformer-based models such as Llama3 or even GPT-4. Agents will generate and evaluate multiple question formats, including multiple-choice and short-answer questions.
  4. Integration and Scalability:
    The system will be deployed using NVIDIA Triton Inference Server to serve the models efficiently, enabling scalability and low-latency responses. The development pipeline will incorporate Dockerized NVIDIA NGC containers for streamlined experimentation and deployment.

Expected Results

Our project aims to deliver a fully functional system for automated grading and question generation before May 2025. This system will incorporate advanced OCR models, multi-agent grading, and question generation frameworks, with a particular focus on Arabic handwritten text. The anticipated outcomes include a detailed research paper, which will be submitted to a leading AI or educational technology conference for publication.
